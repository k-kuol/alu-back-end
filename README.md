# ALU Back-End - API

## Description
Python scripts that interact with a REST API to gather and export employee TODO list data.

## Tasks

### 0. Gather data from an API
Displays an employee's TODO list progress.
```
python3 0-gather_data_from_an_API.py <employee_id>
```

### 1. Export to CSV
Exports an employee's TODO list to a CSV file (`USER_ID.csv`).
```
python3 1-export_to_CSV.py <employee_id>
```

### 2. Export to JSON
Exports an employee's TODO list to a JSON file (`USER_ID.json`).
```
python3 2-export_to_JSON.py <employee_id>
```

### 3. Dictionary of list of dictionaries
Exports all employees' TODO lists to `todo_all_employees.json`.
```
python3 3-dictionary_of_list_of_dictionaries.py
```
